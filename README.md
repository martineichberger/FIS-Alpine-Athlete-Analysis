# FIS-Alpine-Athlete-Analysis

Version v4.1 erweitert den Reiter **Athletendaten**.

Direkt aus der FIS-Athletenseite werden nun ausgelesen:
- Nachname
- Vorname
- Geburtsdatum
- Nation
- Nationenflagge
- Alter
- Geschlecht
- Verein

Lokal starten:
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
